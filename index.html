Additionally, utilized a PUB/SUB system to handle over two million background messages per day to keep the system synchronized with Tipti's main databases in a functional manner utilizing advanced asynchronous logic.
  • Stock Manager Consumer System: Developed a centralized stock information manager Python automated batch process to asynchronously gather stock information from relevant stores' and process all necessary database changes in Tipti's system in a fast, maintainable, and scalable fashion. This software allowed the company to speed up its stock information synchronization from once every two hours to once every ten minutes.
  • AI-Enhanced Checkout Recommendations: Leveraged generative artificial intelligence in conjunction with a Flask API to detect shopper intention in small orders and proceed to generate relevant recommendations that could enhance the shopping experience on checkout and increase potential upselling for the company. This model employs a state-of-the-art XGBoost data mining model for feature selection, optimization, and training to ensure the best possible predictions.
  • Data Mining models performance comparison in a substance abuse classification problem: Performance and runtime comparison between Random Forest prediction models and Multilayer Perecptron prediction models.
